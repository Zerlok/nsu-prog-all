Данный проект содержит код симуляции и восстановления изображений методами фурье-птихографии
для курсовой работы на кафедре АФТИ 3 курс ФФ НГУ.

Для генерации набора изображений низкого разрешения (симуляция сбора данных с микроскопа)
используйте generator.py
Для запуска процесса восстановления изображения исползуйте recoverer.py

Также можно использовать bash скрипты, которые были написаны для сбора данных о симуляции
и востановлении для исследования методов фурье-птихографии для курсовой работы.
